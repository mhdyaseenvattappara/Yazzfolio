# **App Name**: Vanguard Folio

## Core Features:

- Interactive Hero Section: A dynamic hero element that responds to user interactions.
- Project Showcase: Display projects in a grid layout with high-quality thumbnails.
- Sticky Navigation: A header that subtly adjusts upon scrolling.
- Animated Transitions: Micro-interactions and smooth transitions for enhanced user experience.
- Skills Visualization: Showcase skills using animated charts.
- Contact Form with Animated Feedback: Allow users to send messages, accompanied by success/error feedback with an animation.
- Mode Toggle: Enable a light/dark mode toggle to switch between light and dark color schemes

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to highlight interactive elements and links. A vibrant accent choice conveying innovation and modernity against a dark background.
- Background color: Deep charcoal (#1E293B) to create a minimalist and modern feel. Provides a sophisticated backdrop that makes content pop.
- Accent color: Soft grayish-blue (#94A3B8) to complement the electric blue. Subtly guides the user's eye without overpowering the primary call to action.
- Font: 'Inter', a grotesque-style sans-serif suitable for both headlines and body text, for a modern, neutral look. Note: currently only Google Fonts are supported.
- Employ a two- or three-column grid layout for project thumbnails to present content in a structured and visually appealing manner.
- Incorporate a staggered scroll-reveal animation for content sections to enhance engagement and visual interest.
- Implement a 3D-tilt effect and smooth color/shadow transitions on project cards upon hover to encourage interaction and provide visual feedback.
